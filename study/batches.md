## How to run a batch

1. Open `GH-300.pdf` and answer from memory or cover the answer block until you commit.
2. Reveal the PDF’s “Correct Answer” lines only after attempting each item.
3. Record every miss in [misses-log.md](./misses-log.md) with the batch id and topic.
4. For any miss, add the matching link from [docs-verification.md](./docs-verification.md) to your queue.
