# Spaced repetition for GH-300 misses

Transfer question numbers from [misses-log.md](./misses-log.md) into the queues below. Re-quiz from the PDF (stem only, no peeking at answers) until stable.

## Schedule

| Queue | When to review | Question IDs | Done |
| --- | --- | --- | --- |
| R1 | 1 day after first miss | | ☐ |
| R2 | 3 days after R1 pass | | ☐ |
| R3 | 7 days after R2 pass | | ☐ |

## How to use

1. When you miss Q*n*, add *n* to **R1**.
2. On the R1 day, retry only those items. If still wrong, keep *n* in R1 and add a short note in the misses log.
3. When R1 is clean for *n*, move *n* to **R2**, then **R3** using the same rule.
4. After **R3** pass, remove *n* from active queues (optionally mark “retired” in the misses log).
